Import linked Media from an other URL to your WordPress Media Library

== Description ==
Did you copy-paste some content from an old site? Or from an other site, which you fear will be shut down some day? Use this Plugin to import Images, Documents, Audio-Files, Movies and much more to your Media-Library and automatically rewrite URLs to your actual Site

== Installation ==
Use WordPress\' Add New Plugin feature, searching \"Media Importer\", or download the archive and:

1. Unzip the archive on your computer
2. Upload duplicate-post directory to the /wp-content/plugins/ directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Go to Settings -> Import Media and customize behaviour as needed
